# Entrega Final – Ecosistema de Automatización IA

Proyecto desarrollado para el curso **Automatización e Inteligencia Artificial** de Coderhouse.

## 📌 Objetivo

Automatizar el proceso de clasificación de leads comerciales utilizando Make, OpenAI, Notion y Gmail.

## 🛠 Tecnologías utilizadas

- Make
- OpenAI GPT
- Notion
- Gmail
- JSON

## ⚙️ Flujo de automatización

1. Detección de un nuevo lead en Notion.
2. Análisis del lead mediante OpenAI.
3. Procesamiento de la respuesta en formato JSON.
4. Actualización automática del registro en Notion.
5. Envío de un correo electrónico para revisión humana.
